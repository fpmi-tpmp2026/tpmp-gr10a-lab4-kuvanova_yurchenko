## Project Name
**Music Salon** - Консольное приложение для автоматизации работы музыкального салона

## Description
Приложение предназначено для управления музыкальным салоном, который продает компакт-диски. Система позволяет вести учет компакт-дисков, треков, операций поступления и продажи. Приложение поддерживает аутентификацию пользователей с разделением ролей (администратор/пользователь), формирование аналитических отчетов.
### Описание проекта
Консольное приложение для управления музыкальным салоном. Система позволяет:
- Управлять компакт-дисками и треками
- Оформлять операции поступления и продажи
- Формировать аналитические отчеты

### Установка и запуск (Windows)

### Требования
- Установленный MinGW (g++)
- Установленный SQLite3

### Шаги для сборки и запуска

```bash
# Создание папок
mkdir build bin

# Компиляция (каждый файл отдельно)
g++ -c src/Database.cpp -Iinclude -o build/Database.o
g++ -c src/Auth.cpp -Iinclude -o build/Auth.o
g++ -c src/Queries.cpp -Iinclude -o build/Queries.o
g++ -c src/Functions.cpp -Iinclude -o build/Functions.o
g++ -c src/main.cpp -Iinclude -o build/main.o

# Линковка
g++ build/*.o -o bin/music_salon.exe -lsqlite3

# Запуск
cd bin
music_salon.exe
```
